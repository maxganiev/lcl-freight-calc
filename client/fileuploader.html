Бриф по загрузке таблиц

  • Ниже приведена подробная инструкция по составлению таблиц в такой формат, чтобы их можно было динамически обновлять и подгружать в базу данных. Если вы не хотите формировать таблицы с нуля, то можно скачать существующие образцы и в дальнейшем работать с ними.
  • На текущий момент присутствуют прайс-листы 4-х типов: прайсы на услуги до границы для ЖД и авиаперевозок, и прайсы на услуги после границы для грузов,также прибывающих ЖД транспортом и по воздуху
  • На текущий момент доступны прайс-листы для следующих направлений:
    • ЖД Китай (Север) - Московская обл.
    • ЖД Китай (Центр, Юг) - Московская обл.
    • Авиа Китай - Москва, СПб
    • Авиа Корея - Москва, СПб
  • Подгрузка таблиц построена таким образом, что система считывает название загружаемого файла, и по нему автоматически заливает цены в базу данных, которая носит соответствующее название.
  • Наименование таблиц собирается по следующей логике:
    • Для услуг "до границы":
    • тип операции (exp - цены до границы)_ тип доставки _ код страны, откуда осуществляется доставка _ город /регион отправки (при наличии), например: exp_rail_cn_north
    • Для услуг "после границы":
    • тип операции (imp - цены после границы)_ тип доставки , например: imp_rail
  • Разрешенные имена таблиц: exp_rail_cn_north (Китай ЖД, Север), exp_rail_cn_center_south (Китай ЖД, Центр/ Юг), exp_air_cn (Китай авиа), exp_air_kor (Корея авиа), imp_rail (локальные расходы на станции Ворсино/ Электроугли), imp_air (локальные расходы в аэропортах прилета Мск, СПб)
  • Образцы таблиц для заполнения можно скачать тут
  • Обратите внимание, что загружать необходимо таблицы без шапок , т.е. только фактические данные. Образцы таблиц можно посмотреть в файле sample.xlsx
